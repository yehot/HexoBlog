# HexoBlog

My Hexo Blog 文章仓库

## Hexo Doc

https://hexo.io/zh-cn/docs/server

## Usage

本地预览：

```
hexo s
hexo server
```

新建一篇文章

```
hexo new "post title with whitespace"
```

生成静态文件

```
hexo g
hexo generate
```

部署网站

```
hexo d
hexo deploy
```
